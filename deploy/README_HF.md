# 🎯 Pleadly — 聪明地求职

AI招聘经理视角的全流程求职助手。粘贴简历和JD，一键获取从投递到入职的完整分析报告。

## 功能
- ATS关键词检测
- 岗位匹配度评分
- JD全维度拆解(含公司背景联网研究)
- 简历逐行诊断
- 差距学习计划
- 全流程面试模拟题库
- 联网搜索真实面经

## 环境变量
在 Hugging Face Spaces 的 Settings → Secrets 中配置:
- `DEEPSEEK_API_KEY`: 你的 DeepSeek API Key
